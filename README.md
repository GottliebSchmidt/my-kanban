# Kanban Board React + TypeScript

A simple Kanban board application built with React and TypeScript.
Allows drag-and-drop of tasks between columns, adding, editing, and deleting tasks and columns.
State is saved to localStorage for persistence.

## Technologies Used

- React 18
- TypeScript
- React DnD (drag and drop functionality)
- Vite (build tool and dev server)
- CSS for styling

## Features

- Drag and drop tasks between columns
- Add new tasks and columns
- Edit and delete tasks
- Delete columns (with confirmation)
- Persistent state saved in browser localStorage
- Responsive and user-friendly interface

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
npm install
npm run dev
